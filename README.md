## Описание проекта

Это графическое приложение (GUI) для конвертации валют. Программа использует внешний API для получения актуальных курсов и позволяет сохранять историю операций.

**Основные возможности:**
*   **Конвертация:** Перевод суммы из одной валюты в другую.
*   **Выбор валют:** Выпадающие списки для выбора исходной и целевой валюты.
*   **История:** Таблица с историей всех выполненных операций.
*   **Сохранение:** История сохраняется локально в файл `data/history.json`.
*   **Валидация:** Проверка на корректность введенной суммы (должно быть положительное число).
*   **Оффлайн-режим:** История сохраняется и загружается даже без интернета (хотя для новой конвертации интернет нужен).

## Как получить API-ключ (Инструкция по использованию API)

Для этого проекта используется бесплатный сервис **ExchangeRate-API**.
*   **Особенность:** Для бесплатного плана не требуется API-ключ для базового доступа. Код использует публичный эндпоинт с базовой валютой EUR.
*   **Как это работает:** Программа отправляет запрос на получение всех курсов относительно Евро (`EUR`). Затем она пересчитывает вашу сумму через Евро.
*   **Лимиты:** Бесплатный тариф имеет лимиты на количество запросов в месяц. Для коммерческого использования или увеличения лимитов потребуется регистрация и получение персонального ключа на сайте `exchangerate-api.com`.

## Примеры использования и тестирование

### 1. Базовый сценарий (Позитивный тест)
1.  Запустите приложение (`python app.py`).
2.  Оставьте настройки по умолчанию: **Из** `USD`, **В** `EUR`.
3.  Введите сумму `100`.
4.  Нажмите кнопку «Конвертировать».
5.  **Ожидаемый результат:** В поле результата появится эквивалент суммы в Евро. Операция добавится в таблицу истории.
6.  Нажмите «Сохранить историю». Файл `data/history.json` обновится.
7.  Закройте и снова откройте программу — история должна загрузиться автоматически.

### 2. Сценарий с некорректным вводом (Негативный тест)
1.  Запустите приложение.
2.  Введите в поле суммы текст `abc` или отрицательное число `-50`.
3.  Нажмите «Конвертировать».
4.  **Ожидаемый результат:** Появится окно с предупреждением «Сумма должна быть положительным числом!». Операция не выполнится.
